# How to Create a React Application

1. Locate the folder you want to create your React app inside of.

2. Open a terminal, write "cd" then leave a space and then drag and drop the folder in which you want to create your React app.

3. Press enter to get inside the folder.

4. Either use documentations or simply run "npm create vite@latest" to create a React app

5. Give it any name you want, select React as framework and JavaScript as language.

6. Skip the experimental option

7. When it asks "Install and run now?" select No and press enter

8. Then inside the folder, you will find a new folder which is your project

9. Drag and drop that folder on your VSCode, open up your terminal, run "npm install" then "npm run dev"

10. Command + click on the link it gives you to access the webpage

11. You are good to go!
